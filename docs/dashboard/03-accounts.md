## 功能说明

**我的账户** 页面对应 Polymarket 个人资料视图，并承担 **多账户钱包管理** 职责。

### 页面组成

| 区域 | 内容 |
|------|------|
| 账户卡片列表 | 全部账户摘要、切换、编辑 |
| 资料卡片 | 头像、用户名、Proxy 地址、Polymarket 链接 |
| PnL 图表 | 1d / 1w / 1m / 1y / ytd / all 盈亏曲线 |
| 引擎统计 | 今日跟单笔数、成交额、已实现 PnL |
| 持仓 / 成交 | Polymarket 链上持仓、引擎持仓、近期成交 |

---

## 添加账户

**路径：** 我的账户 → **添加账户**

| 字段 | 说明 |
|------|------|
| 账户 ID | 字母、数字、`_`、`-`；创建后 **不可修改** |
| 显示名称 | 侧边栏与列表中的展示名 |
| Proxy 地址 | Polymarket 钱包地址（`0x` + 40 位 hex） |
| 私钥 | 64 位 hex；写入 `.env`，界面不回显 |
| 签名类型 | Proxy ≠ EOA 时选 **1 — Polymarket Proxy** |

### 环境变量命名规则

| 账户 ID | 私钥变量示例 |
|---------|-------------|
| `default` | `POLYMARKET_PRIVATE_KEY` |
| `sub1` | `POLYMARKET_PRIVATE_KEY_SUB1` |
| `main` | `POLYMARKET_PRIVATE_KEY_MAIN` |

创建成功后引擎自动重载，并切换到新账户。

---

## 编辑账户

**路径：** 账户卡片 → **编辑**

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| 显示名称 | 否 | 随时可改 |
| 启用 / 禁用 | 否 | 禁用后该账户不参与跟单循环 |
| Proxy 地址 | 否 | 更换 Polymarket 地址 |
| 新私钥 | **留空则不修改** | 填写则轮换钱包，写入 `.env` |
| 签名类型 | 否 | 留空则保持当前 / 自动推断 |

> 仅改名称或启停状态时，**无需填写私钥**。

---

## 切换账户

两种方式：

1. **侧边栏** 顶部账户下拉框
2. **我的账户** 卡片上的「切换到此账户」

切换后整页刷新，后续 API 请求均指向新账户。

### 多账户隔离范围

| 隔离项 | 说明 |
|--------|------|
| Leader 列表 | 每账户独立配置 |
| SQLite 数据库 | `data/accounts/{id}/` 或 legacy 路径 |
| 今日统计 / 持仓 | 按账户分别计算 |
| 钱包 / 私钥 | 各自 `.env` 变量 |

## PnL 与持仓说明

### PnL 图表

- 优先从 Polymarket `user-pnl-api` 拉取
- 外网不可达时回退到引擎今日统计
- 若图表空白，请检查 **设置 → 网络** 代理

### 三个持仓视图

| 标签 | 数据来源 | 用途 |
|------|----------|------|
| Polymarket 持仓 | 链上 API | 与官网一致的真实持仓 |
| 引擎跟踪 | 本地 SQLite | Preview 模拟 + 跟单记录 |
| 近期成交 | Polymarket 活动 | 钱包最近成交 |

Preview 模式下 Polymarket 持仓可能为 $0，但 **引擎跟踪** 会有模拟记录，属正常现象。
